# Rotating LED Display (POV Display)

An Arduino Nano-based Persistence of Vision (POV) display that creates floating text, numbers, and patterns using 3528 LEDs, a Hall Effect Sensor, and a rotating mechanism. The project demonstrates embedded systems, sensor interfacing, and real-time LED control.

## Features

* POV (Persistence of Vision) technology
* Arduino Nano controlled system
* Hall Sensor synchronization
* Customizable text and patterns
* Low-cost hardware implementation
* Real-time display generation

## Components Used

* Arduino Nano
* 3528 LEDs
* Hall Effect Sensor
* DC Motor
* Power Supply
* Rotating Arm Assembly

## Working Principle

The LEDs are mounted on a rotating arm driven by a DC motor. A Hall Effect Sensor detects a reference position during every rotation. The Arduino Nano uses this signal to synchronize the LEDs and display characters or patterns at precise locations. Due to persistence of vision, the human eye perceives a complete image floating in the air.
